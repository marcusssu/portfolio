Web App & Javascript Development

To-do-list App

2015 Individal Project

EasyNote is a rebuilt web application of the native app CLEAR, I developed and added some useful features upon on it and and made it works on web browser.

Design Cycle

HTML Structure

Designing the HTML & CSS not only for its appearance, but also as an appropriate structure for the javascript to retrieve and manipulate.

Hand Guesture

Hammer.js is a excellent library that allows web browser to detect hand gesture on mobile device, in this project I used drag and swap detection to capture user's input and make corresponding reaction to each gesture.

Calendar

One of the new features I added in EasyNote is that it has a calendar (CLNDR.js) view for people to plan their task ahead of time, and give a hint (the orange line) under the day number.

Data Persistence

Lastly, I used the sklad.js for data persistence, some tricks were used in here to store the complex data structure. I convert all data to a string and store in indexedDB(sklad.js) and convert it back when the user open it the next time.
