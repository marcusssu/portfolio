Design Cycle

Branding & Style Guide

We start with defining the brand and establish the style guide, find out the colour pattern to match with Juu's material, than make essential elements in the web page.

Site Map

We use the site maps / wireframes to organize web site's content, list out all the functions and providing different purchasing flow for account user and new visitor.

HTML & CSS

Implement all the previous design into the HTML & CSS, we did a lot of work to make it responsive and cross browser. For learning purpose we did not use any framework or guide system.
